# ☕ Java Fundamentals Lab

Bienvenidos a mi laboratorio de fundamentos de **Java**. Este repositorio es una bitácora de mi aprendizaje y dominio del lenguaje, cubriendo desde los pilares de la Programación Orientada a Objetos (POO) hasta las características modernas de Java 17+.

> "No se puede construir un gran edificio sobre cimientos débiles". Aquí es donde fortalezco mis bases para el ecosistema Spring.

## 🚀 Contenido del Repositorio

El proyecto está organizado por paquetes temáticos para facilitar la navegación:

- **`poo`**: Aplicación de Herencia, Polimorfismo, Interfaces y Clases Abstractas.
- **`collections`**: Uso avanzado de Listas, Sets y Maps.
- **`exceptions`**: Estrategias de manejo de errores y excepciones personalizadas.
- **`java8`**: Implementación de Programación Funcional, Lambdas y Streams API.
- **`generics`**: Creación de clases y métodos reutilizables con tipos genéricos.

## 🛠️ Tecnologías utilizadas
- **Java SE 17+** 
- **Maven** 
- **JUnit 5** 

## 📖 Cómo usar este repo
Cada paquete contiene una clase `Main` o un conjunto de **Tests Unitarios** que demuestran el concepto.
1. Clona el repositorio: `git clone git@github.com:[TU_USUARIO]/java-fundamentals.git`
2. Compila el proyecto: `mvn clean compile`
3. Ejecuta los tests: `mvn test`
